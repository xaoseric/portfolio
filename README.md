This is the code for my personal site based on vuejs and vuetify.

## Build Setup
*  Install Dependencies

    `yarn install`

* Compiles and hot-reloads for development

    `yarn serve`

* Compiles and minifies for production

    `yarn build`

* Lints and fixes files

    `yarn lint`
